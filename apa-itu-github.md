# Apa Itu Github ?

![](./images/apa-itu-github.png)

GitHub adalah platform berbasis cloud dan situs jejaring sosial untuk pengembang perangkat lunak yang berfungsi sebagai tempat penyimpanan (hosting) repositori Git secara daring. GitHub mempermudah kolaborasi, manajemen proyek, pelacakan perubahan kode, dan version control menggunakan Git, menjadikannya sarana utama bagi developer untuk berbagi dan membangun proyek open-source maupun pribadi. 

Berikut adalah poin-poin penting tentang GitHub:
- Hosting Kode & Git: GitHub menyediakan antarmuka web untuk menyimpan repositori Git, yang memungkinkan pelacakan perubahan kode secara efisien.
- Kolaborasi Tim: Fitur seperti pull request, code review, dan issue tracking memungkinkan pengembang bekerja sama dalam satu proyek tanpa merusak kode satu sama lain.
- Portofolio Developer: Pengguna dapat memamerkan proyek, membangun personal branding, dan terhubung dengan developer lain di seluruh dunia.
Fitur Utama: Mendukung version control, kolaborasi real-time, integrasi CI/CD, dan wiki.
- Milik Microsoft: Platform ini diakuisisi oleh Microsoft pada tahun 2018 dan menjadi layanan terbesar di dunia untuk pengembangan perangkat lunak.
